파이썬 컴프리헨션

컴프리헨션

  • for 루프를 한줄로 표현할수 있는 문법
  • 코드량이 적어져 손가락 건강에 좋다
  • 컴프리헨션은 파이썬 인터프리터에 최적화 되어있어서 속도가 빠르다
  • 컴프리헨션으로 간단하게 리스트, 딕셔너리, 세트를 만들수 있다

List 컴프리헨션

보통의 리스트 루프

리스트 컴프리헨션으로 표현

이렇게 한줄이면 된다. foo(var)는 리스트에 append() 될 값. for var in origin_list는 기존의 for문과 동일하게 생각하면 된다.

딕셔너리를 이용해서 컴프리헨션을 만들수도 있다.

Dictionary 컴프리헨션

딕셔너리도 컴프리헨션으로 만들수 있다.

리스트 컴프리헨션과 그리 다르지 않다. 다른점이라면 []로 둘러쌓인 문법이 {}으로 변한것뿐. for k, v in origin_list.items()는 일반적인 딕셔너리의 루프문과 동일하다. values(), keys()도 사용 가능하다.

조건문

컴프리헨션에 조건문을 걸 수 있다.

기존 컴프리헨션 문법은 동일하게 사용하면서 뒤쪽에 if 비교대상 비교연산자 비교대상 을 넣으면 된다.

Set 컴프리헨션

세트도 컴프리헨션으로 만들 수 있다.

복잡한 컴프리헨션

헤드 퍼스트 파이썬의 예제를 가져와서 복잡한 컴프리헨션을 구현해본다.

거꾸로 읽어나가면 되긴 하는데, 이해가 가는듯 하면서도 어렵다. 마지막의 for dest in set(flights.values())는 리스트 컴프리헨션을 실행하는것 같은데 dest는 그 이전의 리스트 컴프리헨션 안에 쌓여있다. 컴프리헨션 문법 자체가 뒤쪽부터 평가하고, for 앞의 변수를 앞쪽 컴프리헨션에 전달하는것 같은데 정확한 원리는 더 찾아보고 업데이트 해야겠다.
