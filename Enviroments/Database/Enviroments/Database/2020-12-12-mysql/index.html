MySQL 클라이언트 설치

윈도우와 맥에서 늘 쓰던 MySQL Workbench를 사용하려고 했으나, Dbeaver를 쓰기로 했다.

그리고 늘 하던대로 MySQL을 연결했다.

근데 Access denied for user 'root'@'localhost' 메시지가 뜨면서 연결이 안된다. 아니 이게 무슨일이지. 예전엔 MySQL Workbench도 그렇고, Dbeaver도 그렇고, root로 연결이 잘 됐는데 왜 이런오류가 발생하는지 모르겠다.

그래서 검색을 좀 해보니, MariaDB 및 MySQL에서는 별도의 사용자를 생성하고 생성 된 모든 데이터베이스에 대한 접근 권한을 부여해야 한다는 결과를 찾았다.

더 검색해보니 어떻게든 Dbeaver나 MySQL Workbench로 root 접속이 가능하게 만들수도 있는것 같았다. 근데 데이터베이스마다 사용자를 만들어서 권한을 부여하는 것이 좀 더 보안에 좋을것 같았다.
